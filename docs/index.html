05 Metodología

Cómo se detectan las anomalías

Enfoque estadístico

Las anomalías se detectan comparando cada valor diario de un contaminante contra el comportamiento histórico de la misma estación y el mismo contaminante.

El proyecto usa líneas de base estadísticas como IQR o Z-score, y puede incluir un modelo de machine learning no supervisado como Isolation Forest.

Una anomalía indica que el valor fue inusual respecto del patrón histórico. No implica conocer la causa del evento.

Del dato crudo al dashboard

  1. Ingesta de datos oficiales
    Registros abiertos del Gobierno de la Ciudad de Buenos Aires.
  2. Limpieza y agregación
    El pipeline en Python procesa, valida y agrega a nivel mensual y anual.
  3. Detección de anomalías
    IQR, Z-score e Isolation Forest sobre la serie de cada estación.
  4. Publicación estática
    El frontend solo lee JSON preprocesados — sin cálculos pesados en el navegador.
